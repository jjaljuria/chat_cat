# Chat Cat

Un simple chat web

## Instalacion

Ejecute el siguientes comando:

Clonar el repositorio
```
git clone https://github.com/jjaljuria/chat_cat.git
```
Instalar dependencias
```
npm install
```

## Comandos
```
/* ejecuta el servidor */ 
npm start

/* ejecuta en modo de desarrollo */
npm run dev
```
